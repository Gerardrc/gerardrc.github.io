What Renewal Vault does

  • Track renewal and expiry dates for documents and subscriptions
  • Organize items into vaults
  • Set reminders and add events to Calendar
  • Attach photos and files to items
  • Export vaults to PDF

Troubleshooting

  • If reminders are not appearing, check iPhone Settings → Notifications → Renewal Vault.
  • If Calendar export is unavailable, check iPhone Settings → Privacy & Security → Calendars or the app’s settings page.
  • If attachments do not appear correctly, reopen the item and try the attachment again.
